# Pabodha Wanniarachchi - Personal Portfolio

A minimal, high-performance personal portfolio showcasing full-stack development projects, technical skills, and academic background. Designed with a sleek, architectural monochrome aesthetic.

## 🌐 Live Demo

[[Vercel URL](https://pabodhadev.vercel.app/)]

## 🚀 Features

- **Modern Tech Stack:** Built with Next.js, React, and Tailwind CSS.
- **Smooth Animations:** Integrated with Framer Motion for elegant slide-ins, page transitions, and interactive UI elements.
- **Fully Responsive:** Adapts seamlessly to mobile, tablet, and desktop viewports using a custom macro-grid background.
- **Dark/Light Mode:** Full theme support with seamless transitions.
- **Functional Contact Form:** Integrated with Formspree for reliable, serverless email delivery.
- **Performance Optimized:** Clean architecture with minimal external dependencies.

## 💻 Tech Stack

- **Framework:** Next.js
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React & React Icons
- **Form Handling:** Formspree

## 🚀 Getting Started Locally

1. **Clone the repository:**
   ````bash
   git clone [https://github.com/yourusername/portfolio.git](https://github.com/yourusername/portfolio.git)```
   ````
2. **Navigate into the directory:**

```Bash
cd portfolio
Install the dependencies:
```

```Bash
npm install
Start the development server:
```

```Bash
npm run dev
Open http://localhost:3000 in your browser to view the site.
```
